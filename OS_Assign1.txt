1. Display current working directory.
-> pwd
2. List all files and hidden files in current directory.
-> ls -a
-> ls -ld .?*
3. Create a directory named documents .
-> mkdir documents
4. Create subdirectories docs , backup , and scripts .
->cd documents
->mkdir docs, backup, scripts
5. Create files:
file1.txt
file2.txt
notes.txt
-> touch file1.txt, file2.txt, notes.txt
6. Copy file1.txt into backup .
-> cp file1.txt backup
7. Move notes.txt into docs .
-> mv notes.txt docs
8. Rename file2.txt to data.txt .
-> mv file2.txt data.txt
9. Delete data.txt .
-> rm data.txt
10. Remove backup directory recursively.
-> rm -r backup
11. Display contents of example.txt .
-> echo "Hello! Welcome to CDAC" > example.txt
cat example.txt
12. Display first 10 lines of /etc/passwd .
-> head /etc/passwd
13. Display last 10 lines of /etc/passwd .
-> tail /etc/passwd
14. Count lines, words, and characters in file.
-> wc -l -w -c example.txt
15. Search word Linux inside file.
-> grep -i "Linux" file1.txt
16. Search recursively for word error inside /var/log .
-> grep -r "error" /var/log
17. Find all .txt files in home directory.
-> find ~ -type f -name "*.txt"
18. Find empty files inside /tmp .
-> find /tmp -type f -size 0k
19. Find files modified within last 2 days.
-> find ~ -datstart -mtime -2 -print

-> ls -lh report.pdf
